# 🛒 AI Ecommerce Platform

A modern full-stack AI-powered ecommerce application built using React, FastAPI, SQLAlchemy, and Razorpay.

---

# 🚀 Features

## 👤 Authentication

- JWT Login & Registration
- Secure password hashing
- Role-based access (Admin/User)

---

## 🛍️ Ecommerce Features

- Product listing
- Product search
- Category filters
- Shopping cart
- Orders management
- Order summary

---

## 🤖 AI Shopping Assistant

- AI chatbot powered by Groq LLM
- Product-aware recommendations
- Smart shopping suggestions
- Conversational ecommerce experience

---

## 💳 Payments

- Razorpay payment integration
- Dynamic order creation
- Secure checkout flow

---

## 👨‍💼 Admin Dashboard

- Add products
- Delete products
- Manage inventory
- Admin-only access

---

# 🧠 Tech Stack

## Frontend

- React
- React Router
- CSS
- React Toastify

## Backend

- FastAPI
- SQLAlchemy
- SQLite
- JWT Authentication

## AI

- Groq API
- Llama 3

## Payments

- Razorpay
